# Troubles

<div class="aphorism-text">

"You trouble me"  <br/>
I announced  <br/>
to myself.  <br/>
  <br/>
"Yes,  <br/>
You  <br/>
Disturb me <br/> 
Burden me  <br/>
Hurt me."  <br/>
  <br/>
Yet all these troubles  <br/>
distract me  <br/>
from me  <br/>
for a while.  <br/>
  <br/>
Then the resident despot  <br/>
asserts a new self, reinvented.  <br/>
  <br/>
A new old story takes me back to nowhere  <br/>
where I may end up as  <br/>
Real me.  <br/>

</div>
