# Constant Needs

<div class="aphorism-text">

I need you  <br/>
please tell me you know  <br/>
please tell me you trust me  <br/>
I need reassurance  <br/>
less bereftness  <br/>
but who am I  <br/>
unreliable  <br/>
at all times  <br/>
I know that much  <br/>
that little  <br/>
please help me  <br/>
I need to reconcile  <br/>
past, present and future  <br/>
I need continuity  <br/>
I need dependability  <br/>
I need honesty  <br/>
please help me  <br/>
now.<br/>

</div>
