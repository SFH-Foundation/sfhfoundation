# Truth of Now

<div class="aphorism-text">

No way out  <br/>
no way in  <br/>
no out  <br/>
no in  <br/>
no sideways  <br/>
no way  <br/>
no space  <br/>
no time  <br/>
I know all this  <br/>
as I have lost everything else  <br/>
as I am in the perpetual now  <br/>
all lost  <br/>
and all found  <br/>
now.  <br/>

</div>
