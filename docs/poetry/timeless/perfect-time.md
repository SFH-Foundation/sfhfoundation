# Perfect Time

<div class="aphorism-text">

Time is ever there  <br/>
emerging from timelessness  <br/>
ever-fresh and anew  <br/>
ever-generous  <br/>
repeat opportunities  <br/>
every time  <br/>
take it now  <br/>
riding to eternity  <br/>
from time  <br/>
to timelessness  <br/>
that is loyalty  <br/>
honesty  <br/>
divine friendship.  <br/>

</div>
